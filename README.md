## How To Run
Esta aplicacion requiere Node y PHP Composer
```
composer update
```
```
npm install
```
```
php artisan key:generate
```
```
php artisan migrate:fresh --seed
```
```
npm run dev
```
Los siguientes comandos se ejecutan por separado:
```
php artisan serve 
```

```
php artisan schedule:work 
```
