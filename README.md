## How To Run
Esta aplicacion requiere Node y PHP Composer
```
composer install | composer update
```
En la en la raiz
```
npm install
```
```
php artisan key:generate
```
```
php artisan migrate:fresh --seed
```
```
php artisan serve 
```
```
php artisan schedule:work 
```
```
npm run dev
```
